# Antra Nakhasi — Portfolio

Personal portfolio site for Antra Nakhasi, built on a fork of the [Millennial](https://github.com/LeNPaul/Millennial) Jekyll theme.

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

The site will be available at [http://localhost:4000](http://localhost:4000).

## Structure

- `_data/settings.yml` — menu items and social links
- `pages/` — content for About Me, Experience, Education, and Projects
- `_layouts/home.html` — landing page tile layout and animations (Lottie avatar, terminal typing loop, experience SVG)
- `_sass/` — styles, with `_-sections-dir.scss` as the import entry point
- `assets/` — images and compiled stylesheets
